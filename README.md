\# 3.4 GHz GaAs FET RF Amplifier Design



\## Overview

This project demonstrates the complete RF amplifier design workflow at 3.35–3.45 GHz using CE3512K2 GaAs FET in Keysight ADS.



The design includes:

\- S-Parameter Analysis

\- Stability Network Design

\- Gain \& Stability Circle Analysis

\- Smith Chart Impedance Matching

\- Microstrip Equivalent Circuit

\- Layout Generation

\- EM Simulation using RFPro

\- Gerber Generation



---



\## Design Specifications



\- Frequency Range: 3.35–3.45 GHz

\- Gain > 14 dB

\- Input Return Loss (S11) < -15 dB

\- Output Return Loss (S22) < -15 dB

\- Device: CE3512K2 GaAs FET



---



\## Design Flow



1\. Imported S2P transistor data

2\. Performed stability analysis

3\. Designed input/output matching networks

4\. Converted to microstrip implementation

5\. Optimized design parameters

6\. Generated layout

7\. Performed EM simulation and co-simulation
